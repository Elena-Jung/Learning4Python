HTML Attribute

HTML Attribute 문서

해당 문건을 실습하는 과정 중, 이미지 소스를 넣는 방법으로 두가지를 제시하고 있습니다.

하나는 다른 사이트에 업로드 되어 있는 이미지 파일의 주소를 가져와 넣는 방법.(Absolute URL)

또 하나로는 웹 사이트 내에 호스팅 되는 이미지에 연결하는 방법을 제시하고 있습니다(Relative URL)

두번째 방법의 경우, 외부 링크를 통한 이미지 구현 방법이 아닌, 물리적으로 해당 이미지를 디스크에서 가져오는 방법입니다.

작성 중 문제 해결 : / 하나를 소스의 제일 앞에 넣어놨었음. 명백한 휴먼 에러.
